Place a royalty-free music file here named exactly: bg_music.mp3

The pipeline mixes it under the voiceover at low volume (MUSIC_VOLUME, default 0.10).
If this file is absent, the video is built with voiceover only (ENABLE_MUSIC is ignored).

Where to get royalty-free / CC0 tracks:
  - YouTube Audio Library (free, attribution sometimes required)
  - Pixabay Music (https://pixabay.com/music/) - CC0-ish, no attribution
  - Incompetech (Kevin MacLeod, CC BY)
  - Free Music Archive (check each track's license)

Do NOT commit copyrighted music. Pick a track you have the right to use.
